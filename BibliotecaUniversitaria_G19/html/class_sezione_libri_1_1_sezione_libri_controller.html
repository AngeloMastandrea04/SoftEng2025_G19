Detailed Description

La classe SezioneLibriController si occupa di gestire tutte le operazioni sulla struttura dati contenente i Libri.

Il Controller contiene la TableView per la lista dei Libri e le relative colonne, un campo di testo per la ricerca e il bottone per cancellare un Libro. Il Controller ha un riferimento alla lista di Libri contenuta nella classe Biblioteca.

◆ cancellaLibro()

void SezioneLibri.SezioneLibriController.cancellaLibro ( )
private

Metodo di cancellazione del Libro. Ha il compito di cancellare il Libro dalla relativa struttura dati.

Precondition
Viene selezionato un Libro dalla lista e viene premuto il bottone "Cancella".
Postcondition
Viene mostrato un alert di conferma e, se i numeri di copie totali e disponibili combaciano il Libro selezionato viene cancellato dalla lista, altrimenti lancia un alert di errore.
