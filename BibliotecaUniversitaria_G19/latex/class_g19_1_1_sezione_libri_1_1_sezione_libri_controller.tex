\doxysubsection{Detailed Description}
La classe \doxylink{class_g19_1_1_sezione_libri_1_1_sezione_libri_controller}{Sezione\+Libri\+Controller} si occupa di gestire tutte le operazioni sulla struttura dati contenente i Libri. 

Il Controller contiene la Table\+View per la lista dei Libri e le relative colonne, un campo di testo per la ricerca e il bottone per cancellare un \doxylink{class_g19_1_1_sezione_libri_1_1_libro}{Libro}. Il Controller ha un riferimento alla lista di Libri contenuta nella classe \doxylink{class_g19_1_1_biblioteca_1_1_biblioteca}{Biblioteca}.
